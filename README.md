# homebrew-ssmctl

Homebrew tap for [ssmctl](https://github.com/rhysmcneill/ssmctl) — a lightweight CLI for AWS SSM connections, remote command execution, and file transfers.

## Install

```bash
brew tap rhysmcneill/ssmctl
brew install ssmctl
```

## Upgrade

```bash
brew upgrade ssmctl
```

## Uninstall

```bash
brew uninstall ssmctl
brew untap rhysmcneill/ssmctl
```

## Formula is auto-updated

The formula in this tap is automatically bumped by the [ssmctl release workflow](https://github.com/rhysmcneill/ssmctl/blob/main/.github/workflows/release-please.yml) whenever a new version is published. No manual intervention is needed.
